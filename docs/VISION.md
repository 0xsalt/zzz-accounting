# Project Vision: ZZZ Accounting

## The Problem (no solutions allowed)

Developers need portfolio pieces that demonstrate UI craft, design taste, and technical execution — but most portfolio sites are either generic templates or over-engineered toys. A developer's portfolio should show they can build something that *feels real* — polished, intentional, and atmospheric — not just technically functional.

## Why Now

New default web stack (Next.js 15 + Tailwind v4 + shadcn/ui + Bun) needs a showcase project. The concept is strong enough to be memorable in a sea of todo apps and weather dashboards.

## Success Headline

"Slick themed site with great UI craft — wait, is this from a movie?"

## Where Intelligence Lives

N/A — deterministic system. Purely static site with no AI component.

## Target User

Developer peers and community members who appreciate both technical craft and creative execution. People who browse portfolio sites, GitHub demos, and design showcases.

## Success Metrics (3 max)

1. A viewer cannot immediately tell it's a demo — it reads as a real firm's site for at least 10 seconds
2. Someone who has seen *The Accountant* catches the references and grins
3. The typography, spacing, and micro-interactions feel intentional — zero "template energy"

## What This Is NOT

- **Not a real accounting tool** — no calculators, invoice generators, or financial logic. It is a brochure site, even if it's a very good one.
- **Not a backend project** — no auth, no database, no server logic. Purely static export.
- **Not an animation showcase** — tasteful micro-interactions only. No scroll-jacking, no parallax overload, no loading spinners for effect.
- **Not mobile-first** — desktop is the showcase. Responsive is nice-to-have, not the design driver.

## Constraints

- Static site (Next.js static export)
- No external API dependencies
- Must deploy to GitHub Pages
- Desktop-optimized viewport (1280px+)
- No runtime JavaScript dependencies beyond React

## The Gut Check

Someone browses to the site. Clean corporate aesthetic. Professional services listed with precise, understated copy. Everything feels legitimate — the kind of firm that handles offshore accounts and doesn't advertise. Then they notice a detail. The address. A name. The tagline. They look closer. *"Wait... is this from The Accountant?"* That's when it lands.

## Anti-Drift Criteria

- If a proposed feature requires a server or database, we've drifted
- If a proposed feature adds a functional accounting tool (calculator, form processor), we've drifted
- If a proposed animation takes more than 300ms or requires scroll-jacking, we've drifted
- If mobile layout work exceeds 20% of total effort, we've drifted
- If the site stops looking like a real firm's site, we've drifted
